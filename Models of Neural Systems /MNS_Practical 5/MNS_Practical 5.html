#Importing the Euler-function from the last exercise sheet:

def euler(f_func, x_0, t_max, dt):
    
    x = x_0
    nsteps = int(t_max / dt)
    xs = np.zeros(nsteps)
    xs[0] = x_0
    i = 1
    t = 0
    
    while i < nsteps:
        
        x = x + f_func(x,t)*dt
        xs[i] = x
        t += dt
        i += 1
    
    return xs

def euler_mod(f_func, x_0, t_max, dt):
    
    x = x_0
    nsteps = int(t_max / dt)
    xs = np.zeros(nsteps)
    xs[0] = x_0
    i = 1
    t = 0
    
    while i < nsteps:
        
        x = x + f_func(x,t)*dt
  
        if x > V_thresh:
            xs[i] = V_reset

        else:
            xs[i] = x
            
        x = xs[i]    
        t += dt
        i += 1
    
    return xs

$\textit{Top-figure}$: The $ISI$ in this case is 20 $ms$ and hence the firing rate is $\frac{1}{20}$ $\frac{spikes}{ms}$. The number of spikes is 5. We can estimate the firing rate by divinding the total time interva by the number of spikes, which would give us the same result. In this case, therefore, the inverse $ISI$ and firing rate correspond (this is as expected). Moreover, nothing changes between the two different time intervals. Using some quick estimations, we can verify that the firing rate is indeed the inverse of the $ISI$. $\textit{Middle-figure}$: As we should expect, the firing rate increases as a function of the applied current. The empirical results from this model decently approximate the analytical solution. $\textit{Bottom-figure:}$ The Mean $ISI$ – again, as expected – decreases as a function of applied current. And again, the empirical results approximate the analytical predictions.

However, this is not how we would expect from a real neuron. The spike intervals in real neurons are limited by the timescales of the molecular processes that e.g. the resting potential. As a consequence, they cannot go the refractory period.
